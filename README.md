# Architecture Roulette

## Description

Architecture Roulette is a simple app that randomly generates a full stack web app architecture in the style of a LAMP/MEAN stack. Use this app to discover libaries and frameworks you haven't worked with before, or explore combinations of such tools.

The app itself was built using React, Tailwind, and Vite.

## Usage

1. Clone the repository and navigate to the project directory.
2. Run `yarn` to install the dependencies.
3. Run `yarn dev` to start the app locall
